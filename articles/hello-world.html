Welcome to my blog. This is the first post, built with a simple static site generator using plain HTML, CSS, and a small Python build script.

Why static?

Static sites are fast, secure, and easy to deploy. No databases, no server-side rendering — just files served from a CDN.

The entire build process uses only Python's standard library. Each article is an HTML fragment that gets assembled into a full page with a shared header, sidebar, and stylesheet.
