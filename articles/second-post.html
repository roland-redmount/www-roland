This is the second article on the site. Adding a new post is as simple as creating an HTML fragment in the articles/ directory and running the build script.

How it works

The build script scans the articles/ folder, reads the title and date from HTML comments at the top of each file, auto-generates the sidebar navigation, and assembles each article into a full page using a shared template.

The result is a set of static HTML files ready to deploy to Cloudflare Pages or any other static hosting provider.
